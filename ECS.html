Beschreibung

Amazon Elastic Container Service (ECS) ist ein vollständig verwalteter Container-Orchestrierung Dienst,

mit dem Sie containerisierte Anwendungen effizienter bereitstellen, verwalten und skalieren können.

Es ist tief in die AWS-Umgebung integriert und bietet eine benutzerfreundliche Lösung für die Ausführung von Container-Workloads in der Cloud und On-Premises mit erweiterten Sicherheitsfunktionen mithilfe von Amazon ECS Anywhere.

  • Container-Orchestrierung: ECS ermöglicht das einfache Bereitstellen, Verwalten und Skalieren von Docker-Containern in der AWS-Cloud.
  • Container-Definitionen: Definition von Containeranwendungen über die Verwendung von Aufgaben (Tasks) und Services.
  • Docker-Integration: ECS unterstützt Docker-Container und ermöglicht die Verwendung von Docker-Images.
  • Task-Definitionen: Spezifikationen, wie eine Aufgabe (Task) in einem Container ausgeführt wird, einschließlich der Definition von Ressourcen, Netzwerken und Volumes.
  • Service-Definitionen: Konfiguration von Services, um langfristig laufende Anwendungen zu erstellen und zu verwalten.
  • Cluster: Die Grundlage von ECS, in dem Container-Instanzen gestartet und Aufgaben ausgeführt werden.
  • Auto Scaling: Automatische Anpassung der Anzahl von laufenden Tasks basierend auf definierbaren Metriken.
  • Load Balancing: Integration mit dem Elastic Load Balancing-Dienst für die Lastenverteilung auf Container.
  • AWS Fargate: Option zur Ausführung von Containern ohne die Notwendigkeit, EC2-Instanzen selbst zu verwalten.
  • Integration mit Amazon ECR: Nahtlose Integration mit dem Elastic Container Registry (ECR) für das Speichern, Verwalten und Bereitstellen von Docker-Images.
  • IAM-Rollen: Verwendung von IAM-Rollen zur Steuerung des Zugriffs auf ECS-Ressourcen.
  • Logging und Überwachung: Integration mit AWS CloudWatch Logs und AWS CloudWatch Metrics zur Protokollierung und Überwachung von Containeraktivitäten.
  • Integration mit AWS App Mesh: Verbindung von Anwendungen über den Service Mesh-Dienst.
  • Plattformversionen: ECS ermöglicht die Verwendung von verschiedenen Plattformversionen für die Unterstützung neuer Funktionen.
  • AWS Copilot: Ein CLI-Tool, das die Entwicklung, Bereitstellung und Verwaltung von Anwendungen auf ECS vereinfacht.
  • AWS CloudFormation-Integration: Bereitstellung und Verwaltung von ECS-Ressourcen über AWS CloudFormation.
  • AWS Copilot: Ein CLI-Tool, das die Entwicklung, Bereitstellung und Verwaltung von Anwendungen auf ECS vereinfacht.
